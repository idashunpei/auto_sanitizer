# README #

Control servo motor by HC-SR04 ultrasonic sensor

Inspired by https://www.facebook.com/groups/mcalglobal/permalink/2920111298032751/

### Pin assignment ###

| inside | outside |
| ------ | ------- |
| 3.3V   | 5V (Vcc)|
| GPIO2  | 5V (Vcc)|
| GPIO3  | GND     |
| GPIO4  | GPIO14  |
| GND    | GPIO15  |
| GPIO17 (SERVO) | GPIO18  |
| GPIO27 (TRIG)  | GND     |
| GPIO22 (ECHO)  | GPIO23  |
| 3.3V   | GPIO24  |
| ...    | ...     |
